# ⚡ Quick Start Guide - 5 Minutes to Your Store

## 🎯 Get Started in 3 Simple Steps

### Step 1️⃣: Sign Up (2 minutes)
1. Go to: `https://bundles-store.vercel.app`
2. Click **"Sign Up"**
3. Fill name, email, phone, password
4. Verify email
5. Login

### Step 2️⃣: Create Store (2 minutes)
1. In dashboard, click **"Create My Store"**
2. Enter store name & description
3. Submit for approval
4. Wait for approval (usually same day)

### Step 3️⃣: Set Prices & Start Selling (1 minute)
1. Go to **"Pricing"** in dashboard
2. Set your prices for data bundles
3. Copy your store link: `https://bundles-store.vercel.app/store/your-name`
4. Share link with customers
5. Start earning! 💰
